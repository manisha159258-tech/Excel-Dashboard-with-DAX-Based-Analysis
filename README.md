Sales & Quantity Analysis Dashboard (Excel & DAX)

This project is an interactive Excel dashboard built using Power Pivot and DAX, designed to analyze sales and quantity data. It provides key insights into top products, regions, sales trends, and distribution performance.

📁 Files in this Repository
Sales_Quantity_Analysis_Dashboard.xlsx — Interactive Excel dashboard (Power Pivot & DAX)
Sales_Quantity_Analysis_Dashboard.pdf — Static snapshot of the dashboard
Sales_Quantity_DAX_Measures.txt — List of all DAX measures used in the Excel dashboard

⚡ Key Features
Top products and regions
Total and average sales, YoY growth
Quantity and unit price trends
Outlet sales and distribution insights
Fully interactive slicers and charts in Excel Desktop

⚠️ Note: Full interactivity requires Excel Desktop 2016 or later. Excel Online may not support Power Pivot or DAX functionality.

▶️ How to Use
Download Sales_Quantity_Analysis_Dashboard.xlsx
Open it in Excel Desktop
Use slicers to filter data and explore charts interactively
Refer to Sales_Quantity_DAX_Measures.txt for formulas used in the dashboard
